# Registeration

User registeration is the first step in accessing Security Rating. Any new user who is new to Security Rating needs to register on our platform.

Earlier, only those users could register whose domains were already scanned by our platform. Now, in the updated version, we need to register users from organisations whose domains have not been scanned by our platform.
